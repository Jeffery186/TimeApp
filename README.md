# 健康提醒小助手

## 程序说明

- 最早需求是自己用的，就撸了一个，有同样需求的朋友们可以下载使用
- 每隔30分钟弹窗提示一次注意休息，从v1.0.1加入了**提示音**，请注意调节自己电脑的音量
- 点击开始按钮才会开始计时
- 支持Windows 7、Windows 10，理论上只要是电脑装有NET Framework v4.6.1及以上的都可以使用
